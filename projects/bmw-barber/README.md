# BMW Barber

Website institucional para barbearia hospedado em https://bmw-barber.netlify.app/

Detalhes:

- Link: https://bmw-barber.netlify.app/
- Stack: ver `package.json` (React, TypeScript, TailwindCSS, Radix UI, Supabase, Vite, entre outras dependências listadas).

Como rodar localmente (exemplo):

1. Entrar na pasta do projeto:

```bash
cd projects/bmw-barber
```

2. Instalar dependências:

```bash
# usar npm ou yarn conforme preferir
npm install
```

3. Rodar em desenvolvimento:

```bash
npm run dev
```

Observação: este package.json contém apenas as dependências utilizadas no projeto; o código fonte (componentes, páginas, estilos) não foi incluído aqui — apenas a listagem de stacks/deps conforme fornecido.
